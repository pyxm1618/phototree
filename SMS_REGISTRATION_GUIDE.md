## 方案 A：SUBMAIL（赛邮）⭐ 最简单

### 优势
- ✅ **无需审核**，注册即用
- ✅ 支持国际短信
- ✅ 有免费测试额度
- ✅ API 最简单

### 注册步骤（5分钟）

#### 1. 注册账号
访问：https://www.mysubmail.com/signup

填写：
- 邮箱
- 密码
- 公司名称（个人填个人姓名即可）

**提示**：不需要营业执照！

#### 2. 实名认证
登录后 → 右上角头像 → 账户设置 → 实名认证

需要：
- 身份证正反面照片
- 手机号验证

**审核时间**：通常 10 分钟内

#### 3. 创建短信应用
左侧菜单 → 短信 → 应用管理 → 创建应用

填写：
- **应用名称**：PhotoTree
- **应用简介**：照片树项目

#### 4. 创建短信模板
左侧菜单 → 短信 → 模板管理 → 新建模板

**模板内容**：
```
【PhotoTree】您的验证码是@var(code)，5分钟内有效。
```

**变量说明**：`@var(code)` 会被替换为实际验证码

**审核时间**：1-30分钟

#### 5. 获取 API 凭证
左侧菜单 → 开发者中心 → API 凭证

需要记录：
- **APP ID**：如 `12345`
- **APP KEY**：如 `abc123def456...`

#### 6. 充值
左侧菜单 → 财务中心 → 在线充值

**建议**：先充 50 元测试（约 1000 条短信）

## 方案 B：阿里云短信

### 注册步骤（15分钟）

#### 1. 注册阿里云账号
访问：https://account.aliyun.com/register

需要：
- 手机号
- 实名认证（支付宝或银行卡）

#### 2. 开通短信服务
访问：https://dysms.console.aliyun.com

点击"立即开通"

#### 3. 创建签名
控制台 → 国内消息 → 签名管理 → 添加签名

填写：
- **签名名称**：PhotoTree
- **适用场景**：验证码
- **签名来源**：网站（需要已备案的域名）

**上传材料**：
- 网站备案截图
- 或企业营业执照

**审核时间**：1-2小时

#### 4. 创建模板
控制台 → 国内消息 → 模板管理 → 添加模板

**模板内容**：
```
验证码${code}，您正在登录PhotoTree，5分钟内有效。
```

**审核时间**：1-2小时

#### 5. 创建 AccessKey
控制台右上角头像 → AccessKey管理 → 创建 AccessKey

**安全提示**：建议创建子账号并授权，不要用主账号 Key

记录：
- **AccessKey ID**
- **AccessKey Secret**

#### 6. 充值
控制台 → 充值中心

**建议**：100元（约2200条）

## 方案 C：腾讯云短信

### 注册步骤（20分钟）

#### 1. 注册腾讯云
访问：https://cloud.tencent.com/register

需要实名认证（微信或银行卡）

#### 2. 开通短信服务
访问：https://console.cloud.tencent.com/smsv2

点击"开始使用"

#### 3. 创建应用
短信控制台 → 应用管理 → 创建应用

填写：
- **应用名称**：PhotoTree
- **应用简介**：用户验证

记录：**SDK AppID**

#### 4. 配置签名
国内短信 → 签名管理 → 创建签名

填写：
- **签名类型**：网站
- **签名用途**：自用
- **签名内容**：PhotoTree

**上传材料**：网站备案证明

**审核时间**：2小时

#### 5. 创建模板
国内短信 → 正文模板管理 → 创建正文模板

**模板内容**：
```
{1}为您的登录验证码，请于{2}分钟内填写。如非本人操作，请忽略本短信。
```

**变量**：{1} = 验证码，{2} = 有效时间

**审核时间**：2小时

#### 6. 获取密钥
访问：https://console.cloud.tencent.com/cam/capi

创建密钥，记录：
- **SecretId**
- **SecretKey**

# 微信商户平台添加分账接收方 - 图文操作指南

## 📋 前提条件
- ✅ 已开通分账功能（您已完成）
- ⚠️ 需要知道 KOL 的 OpenID

---

## 第一步：登录微信商户平台

1. 访问：https://pay.weixin.qq.com
2. 使用管理员账号登录

---

## 第二步：进入分账接收方管理

### 路径：
```
产品中心 → 分账 → 分账接收方管理
```

**具体操作**：
1. 点击左侧菜单 **「产品中心」**
2. 展开找到 **「分账」**
3. 点击 **「分账接收方管理」**

---

## 第三步：添加新的分账接收方

### 点击「新增分账接收方」按钮

会出现一个表单，需要填写以下信息：

### 填写信息：

| 字段 | 填写内容 | 说明 |
|------|----------|------|
| **接收方类型** | 选择 `个人` | 因为是给 KOL 个人分账 |
| **接收方账号** | KOL 的 OpenID | 见下方「如何获取 OpenID」 |
| **接收方关系** | 选择 `合作伙伴` | PARTNER |
| **接收方姓名** | KOL 真实姓名 | 可选，但建议填写 |
| **接收方备注** | 如：KOL_XIAOMING | 方便管理 |

### ⚠️ 重要提示：
- **接收方账号（OpenID）** 必须是用户在您的 AppID (`wx746a39363f67ae95`) 下的 OpenID
- OpenID 格式通常是：`oXXXX` 开头，28位字符串

---

## 第四步：如何获取 KOL 的 OpenID

### 方法一：让 KOL 访问您的网站并登录 ✅（推荐）

1. **发送登录链接给 KOL**
   ```
   https://aiguess.cn
   ```

2. **让 KOL 微信登录**
   - KOL 用微信扫码登录
   - 登录成功后告诉您

3. **在数据库查询 OpenID**
   ```sql
   -- 在 Vercel Postgres 控制台执行
   SELECT openid, nickname, avatar_url 
   FROM users 
   WHERE nickname LIKE '%KOL名字%'
   ORDER BY created_at DESC 
   LIMIT 10;
   ```

4. **复制 openid 字段的值**

### 方法二：通过公众号获取

如果您有绑定的公众号：
1. KOL 关注公众号
2. 在公众号后台查看用户列表
3. 查询该用户的 OpenID

### 方法三：通过客服会话

当 KOL 发起客服会话时，可以在客服消息中看到 OpenID。

---

## 第五步：提交并等待审核

1. 填写完所有信息后，点击 **「提交」**
2. 微信会进行审核（通常几分钟到几小时）
3. 审核通过后，状态会变为 **「正常」**

---

## 第六步：绑定 OpenID 到后台邀请码

审核通过后，在终端执行：

```bash
curl -X POST https://aiguess.cn/api/admin/profit-sharing/add-receiver \
  -H "Content-Type: application/json" \
  -d '{
    "referralCode": "KOL_XIAOMING",
    "openid": "oXXXXXXXXXXXXXXXXXXXXXXXXXXX",
    "sharingPercentage": 10.00
  }'
```

**参数说明**：
- `referralCode`: 在管理后台创建的邀请码
- `openid`: 从数据库查到的 KOL OpenID
- `sharingPercentage`: 分账比例，10 表示 10%

---

## 常见问题

### Q1: 找不到「分账接收方管理」菜单？
**A**: 确保已经开通分账功能，刷新页面重新登录试试。

### Q2: 提示「接收方账号不存在」？
**A**: OpenID 必须是用户在您的 AppID 下的，让 KOL 先登录一次 aiguess.cn。

### Q3: 审核不通过？
**A**: 检查关系类型是否选择正确，建议选「合作伙伴」。

### Q4: 可以添加多少个接收方？
**A**: 通常没有数量限制，但每个订单最多分给 50 方。

## 快速操作示例

假设您要添加 KOL "小明"：

1. **小明登录** → 访问 aiguess.cn 并微信登录
2. **您查数据库** → 找到 openid: `oABC123456789...`
3. **商户平台添加** → 类型：个人，账号：`oABC123456789...`，关系：合作伙伴
4. **等审核通过**
5. **绑定到邀请码** → 执行上面的 curl 命令

完成后，小明推广的用户付费时，佣金会自动打到小明的微信零钱！
